system requirements
minimum system requirements
  • OS: Windows 7,8,10 64 Bit.
  • CPU: Core i3 @2.4 Ghz.
  • RAM: 4 GB.
  • GPU: DirectX11 Intel UHD Graphics 620 or AMD Radeon vega 3.
  • Storage: 3 GB.
recommended system requirements
  • OS: Windows 7,8,10 64 Bit.
  • CPU: Core i5 @2.8 Ghz.
  • RAM: 8 GB.
  • GPU: DirectX11 NVIDIA GeForce GTX 560 or AMD Radeon HD 7870.
  • Storage: 3 GB.
